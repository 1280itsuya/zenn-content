結論から言うと、AIをCIに組むと月¥3,800まで膨らんだ課金は、cacheカウンタ・concurrency・夜間monitorの3点で月¥1,200に固定できる。寄与度はカウンタ上限化が62%、並列爆発の抑制が28%、残り10%が不要トリガ削減だった。

## 1. cache + 日次カウンタで Claude 呼び出しを20回/日に上限化

`actions/cache` にカウンタファイルを永続化し、20回超で `skip` を出力する。

```yaml
- uses: actions/cache@v4
  with: { path: .ai-quota, key: ai-quota-${{ github.run_id }}, restore-keys: ai-quota- }
- id: guard
  run: |
    n=$(cat .ai-quota 2>/dev/null || echo 0)
    if [ "$n" -ge 20 ]; then echo "skip=true" >> $GITHUB_OUTPUT; else echo $((n+1)) > .ai-quota; fi
```

## 2. if: 条件で超過時はAI判定をスキップし通常CIへ退避する fail-safe

上限到達時はAIジョブを丸ごと飛ばし、pytestだけ走らせて緑を維持する。

```yaml
  ai-review:
    needs: quota
    if: needs.quota.outputs.skip != 'true'
    runs-on: ubuntu-latest
    steps: [{ run: python ci/claude_review.py }]
```

## 3. concurrency と matrix制御で並列トークン爆発を1/4に抑える

同一PRの古い実行をキャンセルし、matrixの同時実行を1に絞る。

```yaml
concurrency:
  group: ai-${{ github.ref }}
  cancel-in-progress: true
jobs:
  ai-review:
    strategy: { max-parallel: 1, matrix: { shard: [1,2,3,4] } }
```

## 4. claude_review.py で max_tokens=1024 と誤爆ログを残す

出力上限を固定し、判定とコストを1行で追記する。

```python
import anthropic, json
c = anthropic.Anthropic()
r = c.messages.create(model="claude-opus-4-8", max_tokens=1024,
    messages=[{"role": "user", "content": open("diff.txt").read()[:8000]}])
usage = r.usage
print(json.dumps({"in": usage.input_tokens, "out": usage.output_tokens}))
```

## 5. 夜間monitor jobでAnthropic usageを集計しSlackへ閾値警告する

毎晩23時に当月コストを合算し、¥1,200超で警告を飛ばす。

```yaml
on: { schedule: [{ cron: "0 14 * * *" }] }
jobs:
  monitor:
    runs-on: ubuntu-latest
    steps:
      - run: |
          cost=$(python ci/sum_usage.py)
          [ "${cost%.*}" -gt 1200 ] && \
          curl -s -X POST $SLACK_WEBHOOK -d "{\"text\":\"AI cost ¥${cost} 超過\"}"
```

## 6. ANTHROPIC_API_KEY を90日でローテーションし漏洩を検知する

gitleaksで平文混入を弾き、鍵は四半期ごとに差し替える。

```bash
gitleaks detect --redact --exit-code 1 || { echo "key leak detected"; exit 1; }
gh secret set ANTHROPIC_API_KEY --body "$NEW_KEY"   # 90日ごと
```

3ヶ月運用での誤爆率は1.8%（54件中1件）。鍵ローテーションをcronに乗せれば、課金とセキュリティの両方が無人で固定される。
