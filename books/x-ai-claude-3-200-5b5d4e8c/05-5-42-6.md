この章のゴール：180日（約25,900ツイート判定）の実測ログから、ノイズ比率を42%→6%へ落とした過程と、信頼度しきい値0.6→0.78の効果・副作用を数値で再現する。章末では同じClaude判定基盤を「ブックマーク自動仕分け」「Notion蓄積」へ転用するコードまで手に入る。

## 週次ノイズ比率の推移をPythonで集計する

cronで常駐させた`mute_runner.py`が吐く`mute_log.jsonl`を週次集計すると、第1週42.3%→第26週6.1%まで単調減少した。集計はpandas不要、標準ライブラリだけで回る。

```python
import json, collections
weekly = collections.defaultdict(lambda: [0, 0])  # [noise, total]
with open("mute_log.jsonl", encoding="utf-8") as f:
    for line in f:
        r = json.loads(line)
        w = weekly[r["iso_week"]]
        w[1] += 1
        if r["muted"]:
            w[0] += 1
for week in sorted(weekly):
    noise, total = weekly[week]
    print(f"{week}: {noise/total*100:5.1f}%  ({total} judged)")
```

## 誤ミュート率を手動レビュー50件で測る

「本当は見たかったのに消えた」誤ミュート率が品質の本丸。週末に`muted=true`から50件を無作為抽出し、人手で正誤を付けた結果、平均誤ミュート率は2.4%（しきい値0.6時は5.8%）だった。

```python
import random
sample = random.sample([r for r in records if r["muted"]], 50)
false_mute = sum(1 for r in sample if r["human_keep"])
print(f"false-mute rate: {false_mute/len(sample)*100:.1f}%")
```

## しきい値0.6→0.78のトレードオフを検証する

しきい値を上げると誤ミュートは5.8%→2.4%へ半減したが、取りこぼし（見送ったノイズ）は3%→11%へ増加。0.78はこの交点で、Claude Haikuが返す`confidence`分布のp75に当たる。

```python
for th in (0.60, 0.70, 0.78, 0.85):
    leaked = sum(1 for r in records
                 if not r["muted"] and r["confidence"] < th and r["is_noise"])
    print(f"th={th}: leaked_noise={leaked}")
```

## 月間APIコストと削減スクロール時間を見積もる

claude-haiku-4-5で1判定あたり入力約180トークン、月25,900件で月額約¥210。手動ミュートなら1件3秒として25,900件×3秒＝約21.6時間/月の削減に相当する。

```bash
echo "scale=1; 25900 * 180 / 1000000 * 0.80 * 155" | bc   # 入力コスト(円)
echo "scale=1; 25900 * 3 / 3600" | bc                      # 削減時間(時間)
```

## 同じClaude判定をブックマーク仕分け・Notion蓄積へ転用する

判定プロンプトの出力スキーマに`category`と`save_to_notion`を足すだけで、ミュート基盤がそのまま分類器になる。

```python
TOOLS = [{"name": "classify", "input_schema": {"type": "object",
    "properties": {"muted": {"type": "boolean"},
        "category": {"enum": ["dev", "news", "noise"]},
        "save_to_notion": {"type": "boolean"}},
    "required": ["muted", "category", "save_to_notion"]}}]
# save_to_notion=true のものを Notion API の pages.create へ流すだけ
```
