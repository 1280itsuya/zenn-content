## 収益ゼロが続く本当の理由

ブログを開設してアフィリエイトに登録したのに、収益が一向に発生しない――この状態の原因はほぼ一つだ。**記事を書き続けられていない**ことである。人間が手で書ける記事は月に数本が限界で、それでは検索エンジンに評価される前に息切れする。解決策は「自動投稿」と「アフィリエイトリンク埋め込み」を一本のパイプラインに統合することだ。

## パイプラインの全体像

本章では以下の流れを実装する。

```
キーワード選定 → Claude でドラフト生成 → アフィリ링クを挿入 → はてなブログへ自動投稿
```

使用技術は Python 3.11・Anthropic SDK・A8.net アフィリエイトリンクの3点のみ。月額費用はほぼゼロで運用できる。

## Step 1 ── キーワードをリストで管理する

`keywords.txt` にニッチなロングテールキーワードを1行1件で列挙する。たとえば「Python 初心者 エラー 解決」「Claude API 使い方 無料」のような具体的な検索語だ。月30〜100件あれば十分で、ここを充実させるほど記事量が増える。

## Step 2 ── Claude でドラフトを自動生成する

```python
import anthropic

client = anthropic.Anthropic()

def generate_article(keyword: str) -> str:
    message = client.messages.create(
        model="claude-sonnet-4-6",
        max_tokens=2000,
        messages=[{
            "role": "user",
            "content": (
                f"キーワード「{keyword}」で800字のブログ記事を書いてください。"
                "読者は初心者。見出し付き・具体例あり・行動を促すCTAで締める。"
            )
        }]
    )
    return message.content[0].text
```

ここでのポイントは**プロンプトにCTAの指示を入れる**ことだ。「行動を促す締め」を明示しておかないと、記事末尾がアフィリンクの置き場所を失う。

## Step 3 ── アフィリエイトリンクを自動挿入する

A8.netで取得したリンクを辞書で管理し、本文中のキーワードと照合して差し込む。

```python
AFFILIATE_MAP = {
    "Python 書籍": "https://px.a8.net/svt/ejp?a8mat=XXXXX",
    "Claude API":  "https://px.a8.net/svt/ejp?a8mat=YYYYY",
}

def inject_links(body: str) -> str:
    for keyword, url in AFFILIATE_MAP.items():
        anchor = f'<a href="{url}" target="_blank">{keyword}</a>'
        body = body.replace(keyword, anchor, 1)  # 初出1箇所のみ
    return body
```

`replace(..., 1)` で最初の1箇所だけ置換するのが鉄則だ。同じリンクが連発するとスパム判定される。

## Step 4 ── はてなブログへ自動投稿する

はてなブログは AtomPub API を公開しており、`requests` だけで投稿できる。

```python
import requests, os
from datetime import datetime

def post_to_hatena(title: str, body: str) -> None:
    url = f"https://blog.hatena.ne.jp/{os.environ['HATENA_ID']}/{os.environ['HATENA_BLOG']}/atom/entry"
    xml = f"""<?xml version="1.0"?>
<entry xmlns="http://www.w3.org/2005/Atom">
  <title>{title}</title>
  <content type="text/html">{body}</content>
  <updated>{datetime.now().isoformat()}</updated>
</entry>"""
    requests.post(url, data=xml.encode(),
                  auth=(os.environ['HATENA_ID'], os.environ['HATENA_API_KEY']),
                  headers={"Content-Type": "application/atom+xml"})
```

認証情報は `.env` に逃がし、`python-dotenv` で読み込む。スクリプトにハードコードしないこと。

## 毎朝7時に全自動で動かす

Windows タスクスケジューラに以下を登録すれば完成だ。

```
タスク名: auto_affiliate_post
トリガー: 毎日 07:00
操作:    python C:\auto-money\run_pipeline.py
```

`run_pipeline.py` は `keywords.txt` を1行読み進め、記事生成→リンク挿入→投稿の一連を実行する。30日で30記事が自動で積み上がる。

## 最初の収益が発生するまで

記事30本を超えたあたりから検索流入が始まり、アフィリエイトのクリックが発生し始める。焦って記事を手修正したり投稿を止めたりしないこと。パイプラインを信じて**毎朝1本の自動投稿を継続する**のが、収益ゼロを抜け出す唯一の現実解だ。
