## AIが「仕組み」を変えた理由

副業で稼ぐ最大の壁は、**時間**だ。本業を抱えながら毎日コンテンツを作り、投稿し、修正する――人力では週に数本が限界で、収益が出る前に燃え尽きる。

ところがChatGPT・Claude・Pythonを組み合わせると、この構造が根本から変わる。「考える→書く→投稿する」という作業を自動化し、あなたが寝ている間にもコンテンツが公開され続ける仕組みを作れる。月5万円の収益が目標なら、必要なのは初期構築の数日間だけだ。

---

## 全体像を3層で把握する

自動化パイプラインは次の3層に分かれる。

| 層 | 役割 | 使うもの |
|---|---|---|
| 生成層 | テーマ選定・文章生成 | Claude API / ChatGPT API |
| 投稿層 | 各プラットフォームへの自動公開 | Python + 各サービスのAPI |
| 計測層 | ビュー数・収益の記録と改善 | スプレッドシート / SQLite |

収益はこの3層が連動して初めて生まれる。「書くだけ」で止まると、配信されず、計測もできず、改善ループが回らない。

## 最初の一歩：環境を15分で整える

まずPythonの仮想環境を作り、必要なライブラリをインストールする。

```bash
python -m venv .venv
.venv\Scripts\activate          # Windows
pip install anthropic python-dotenv requests
```

次に`.env`ファイルにAPIキーを書く。

```
ANTHROPIC_API_KEY=sk-ant-xxxxx
```

これだけで「Claude APIを呼び出せる状態」が完成する。

## 最小の自動化スクリプトを動かす

以下は、テーマを渡すと記事の骨子を返す最小スクリプトだ。

```python
import anthropic, os
from dotenv import load_dotenv

load_dotenv()
client = anthropic.Anthropic()

def generate_outline(theme: str) -> str:
    msg = client.messages.create(
        model="claude-sonnet-4-6",
        max_tokens=800,
        messages=[{"role": "user", "content": f"{theme}について、副業初心者向けの記事構成を5点で出力して"}]
    )
    return msg.content[0].text

print(generate_outline("Pythonで自動化副業を始める方法"))
```

`python generate.py`を実行すれば、30秒以内に記事の骨子が出力される。これが「仕組み化」の最小単位だ。

## なぜZennやQiitaから始めるのか

ブログを自前で立ち上げると、集客に半年以上かかる。一方、ZennやQiitaには**すでに読者がいる**。AIが生成した記事をAPIで投稿するだけで、初日からビュー数が計測できる。

実際、テック系AI記事をZennに投稿すると、タグ次第で1記事あたり数十〜数百ビューを短期間で獲得できる。これを毎日自動で繰り返せば、月間数千ビューが現実的な数字になる。
