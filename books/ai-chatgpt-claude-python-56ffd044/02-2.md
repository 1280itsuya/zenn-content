## 第2章　最初の収益を生む「自動投稿×アフィリエイト」の仕組みを作る

副業で最初の1円を稼ぐ最短ルートは、**記事を書く作業を自動化しながら、アフィリエイトリンクで収益を得る**という組み合わせだ。本章では、Claude APIとPythonを使い、ブログ記事を自動生成して投稿するパイプラインを構築する。

---

## 必要なものを揃える

まずA8.netでアフィリエイト提携を申請し、自分のアフィリエイトリンク（例：`https://px.a8.net/svt/ejp?a8mat=XXXX`）を取得しておく。次に、Zennのアカウントを作成してGitHubリポジトリと連携させる。Zennは「GitHubにpushするだけで記事が公開される」ため、自動化の相性が抜群に良い。

```bash
pip install anthropic python-dotenv
```

---

## 記事を自動生成するスクリプト

以下のスクリプトが核心部分だ。Claudeにプロンプトを渡し、アフィリエイトリンク付きの記事を生成する。

```python
import anthropic, os
from dotenv import load_dotenv

load_dotenv()
client = anthropic.Anthropic(api_key=os.environ["ANTHROPIC_API_KEY"])

AFFILIATE_LINK = "https://px.a8.net/svt/ejp?a8mat=XXXX"
TOPIC = "Python初心者向けおすすめ技術書5選"

prompt = f"""
技術系ブログ記事を1000字で書いてください。
テーマ: {TOPIC}
・見出しは##から始める
・具体的な商品名と理由を含める
・記事末尾にこのリンクを自然に挿入: {AFFILIATE_LINK}
・読者が「買いたい」と思える締めくくりにする
"""

message = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=2048,
    messages=[{"role": "user", "content": prompt}]
)

article = message.content[0].text
print(article)
```

---

## Zennへ自動投稿する

生成した記事をMarkdownファイルとして保存し、GitHubにpushするだけで公開される。

```python
import subprocess, datetime

slug = f"auto-{datetime.date.today().isoformat()}"
filepath = f"articles/{slug}.md"

header = f"""---
title: "{TOPIC}"
emoji: "📚"
type: "idea"
topics: ["Python", "プログラミング", "副業"]
published: true
---

"""

with open(filepath, "w", encoding="utf-8") as f:
    f.write(header + article)

subprocess.run(["git", "add", filepath])
subprocess.run(["git", "commit", "-m", f"add article: {slug}"])
subprocess.run(["git", "push", "origin", "main"])
```

このスクリプトをTask Schedulerに登録し、毎朝7時に実行すれば**1日1記事・無人投稿**が完成する。

## 収益が発生する仕組み

読者がZennの記事を読み、末尾のアフィリエイトリンクから商品を購入すると報酬が発生する。初月は数百円が目安だが、記事が蓄積されるにつれてSEO流入が増え、自動的に収益が伸びていく。**作業時間は初期セットアップの2〜3時間のみ**、あとはコードが稼ぎ続ける。次章では、この記事品質をLLMで自動評価して改善するフィードバックループを実装する。
