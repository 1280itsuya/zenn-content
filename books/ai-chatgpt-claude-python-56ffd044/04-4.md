## 「仕組みが仕事をする」状態を作る

副業で稼ぎ続けるために最も重要なのは、**毎日手を動かさなくても収益が動く状態**を作ることだ。そのカギが「投稿→分析→改善」を自動で回し続ける自律サイクルである。

このサイクルは三つのフェーズで構成される。

1. **投稿フェーズ** — コンテンツを自動生成してプラットフォームへ公開する  
2. **分析フェーズ** — ビュー数・クリック率・収益を自動収集して評価する  
3. **改善フェーズ** — 勝ちパターンを抽出し、次の投稿テーマに反映する

---

## 投稿を自動化する

まずはコンテンツ生成から投稿までをPythonスクリプトで一本化する。Claude APIを使って記事を生成し、Zenn CLIでpushするだけで公開が完了する。

```python
# generate_and_publish.py
import anthropic, subprocess

client = anthropic.Anthropic()

def generate_article(topic: str) -> str:
    msg = client.messages.create(
        model="claude-sonnet-4-6",
        max_tokens=2000,
        messages=[{"role": "user", "content": f"Zenn向けに「{topic}」の技術記事をMarkdownで書いてください"}]
    )
    return msg.content[0].text

article = generate_article("Claude APIでZenn記事を自動投稿する方法")
with open("articles/auto_post.md", "w") as f:
    f.write(article)

subprocess.run(["npx", "zenn", "preview"], check=True)
subprocess.run(["git", "add", ".", "&&", "git", "commit", "-m", "auto post", "&&", "git", "push"], shell=True)
```

このスクリプトをWindowsタスクスケジューラで毎朝7時に実行すれば、起きたときには記事が公開済みになる。

## 分析を自動化する

投稿したら数字を拾う。Zenn APIはビュー数を公開しているので、Pythonで定期収集してCSVに蓄積する。

```python
# collect_views.py
import requests, csv, datetime

headers = {"Authorization": "Bearer YOUR_ZENN_TOKEN"}
res = requests.get("https://zenn.dev/api/articles?username=yourname", headers=headers)
articles = res.json()["articles"]

with open("data/views_log.csv", "a", newline="") as f:
    writer = csv.writer(f)
    for a in articles:
        writer.writerow([datetime.date.today(), a["slug"], a["liked_count"], a.get("page_views_count", 0)])
```

このログが溜まることで、「どのテーマが読まれているか」が数値で見えてくる。

## 改善を自動化する――勝ちパターンの抽出と反映

分析結果をClaudeに読み込ませ、次週のテーマを自動提案させる。これが「改善フェーズ」の核心だ。

```python
# winner_extractor.py
import pandas as pd, anthropic

df = pd.read_csv("data/views_log.csv", names=["date","slug","likes","views"])
top = df.groupby("slug")["views"].sum().nlargest(5).to_string()

client = anthropic.Anthropic()
msg = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=500,
    messages=[{"role": "user", "content": f"以下の高view記事スラッグから共通テーマを抽出し、来週の記事テーマ5本を提案してください:\n{top}"}]
)
print(msg.content[0].text)
```

出力されたテーマを`topics.txt`に保存し、翌朝の`generate_and_publish.py`が読み込む。これで**サイクルが閉じる**。

---

## サイクルが閉じると収益が複利で伸びる

最初の1週間は反応がなくて当然だ。しかし3週目以降、高ビュー記事のテーマが次の記事に反映され始めると、閲覧数は指数的に増加する。ZennはGitHub連携でSEOにも強く、記事が蓄積されるほど検索流入が増える。

重要なのは「人間が毎日判断しない」構造を作ることだ。投稿・収集・抽出・次回テーマ設定まで全てスクリプトが担う。あなたが行うのは、週に一度ログを眺めてスクリプトを微調整するだけでいい。

この自律サイクルこそが、AIを使った副業の本質的な強みである。
