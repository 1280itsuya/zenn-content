## 半年の実測ログ：7時間/週 → 2.8時間/週

運用開始日から24週間、`logs/usage.jsonl` に手作業の情報収集時間を記録し続けた。週次で集計するスクリプトがこれ。

```python
import json, statistics
from collections import defaultdict

weekly = defaultdict(float)
with open("logs/usage.jsonl") as f:
    for line in f:
        r = json.loads(line)
        weekly[r["iso_week"]] += r["manual_minutes"] / 60

vals = list(weekly.values())
print(f"初月平均: {statistics.mean(vals[:4]):.1f}h/週")   # 7.0
print(f"直近平均: {statistics.mean(vals[-4:]):.1f}h/週")  # 2.8
print(f"削減: {7.0 - 2.8:.1f}h/週 = {(7.0-2.8)*4:.0f}h/月")
```

差分は4.2時間/週。要約配信を読むだけで朝のXタイムラインを開かなくなったのが最大要因で、可処分時間が月16.8時間増えた。

## 月¥580の内訳：Claude Haiku が費用の8割

API費は OpenAI ではなく Claude Haiku に寄せてコストを圧縮した。1日120ポスト要約で月¥580、内訳はこう出る。

```python
COST = {"haiku_in": 0.8/1e6, "haiku_out": 4/1e6}  # USD/token
daily = 120 * (450 * COST["haiku_in"] + 180 * COST["haiku_out"])
print(f"月額: ${daily*30:.2f} ≈ ¥{daily*30*155:.0f}")  # ≈ ¥580
```

Playwright のスクレイピングは無料枠で完結するため、課金はLLM部分のみ。Sonnetに切り替えると月¥4,200まで跳ねるので、要約はHaiku固定にしている。

## Playwright凍結0回：間隔ジッターの実装

運用2週目に1回だけ一時制限を食らった。原因は固定12秒間隔の機械的アクセス。`asyncio` でジッターを入れて以降、22週間で凍結0回。

```python
import asyncio, random

async def fetch_with_jitter(page, urls):
    for url in urls:
        await page.goto(url, wait_until="networkidle")
        await asyncio.sleep(random.uniform(18, 47))  # 平均32秒
```

平均32秒・最大47秒のばらつきが境界線だった。15秒未満に詰めると週内に必ず制限がかかる、というのが半年分の失敗ログから出た閾値。

## note下書きへの転用：要約を有料記事の素材に

収集済みデータは副業に流用できる。`summaries.jsonl` から特定タグを抽出し、note記事の下書きMarkdownを生成する。

```python
import json

rows = [json.loads(l) for l in open("data/summaries.jsonl")]
picks = [r for r in rows if "AI副業" in r["tags"]][:5]

with open("draft_note.md", "w", encoding="utf-8") as f:
    f.write("# 今週のAI副業トレンド5選\n\n")
    for i, r in enumerate(picks, 1):
        f.write(f"## {i}. {r['headline']}\n{r['summary']}\n\n")
```

週5本ネタ × 月4週で20本分の素材が自動で溜まる。手動なら1本30分かかる調査が、配信ログのフィルタだけで済む。

## Task Schedulerで毎朝7時に常駐化

最後にWindowsで無人化する。`schtasks` で毎朝7時に要約配信を起動するワンライナー。

```bash
schtasks /Create /TN "X_Mute_Digest" /TR \
  "powershell -File C:\auto\run_digest.ps1" \
  /SC DAILY /ST 07:00 /RL HIGHEST
```

やってはいけない自動化も明記する。自動フォロー・自動DM・自動いいねはこのリポジトリに一切含めない。半年で凍結0だったのは「読み取り専用」を厳守したから。書き込み系を足した瞬間に境界線を越える。
