package.jsonのpeerDeps解消・ESLint 9のflat config移行・Prettierとの競合解消──全部「調べる」に時間を溶かした。翌月、Claude APIに同じ作業を丸投げした結果がこれだ。

```bash

2024-12-08 10:30:44 +0900 chore: add eslint flat config
2024-12-08 10:02:17 +0900 initial commit
```

差は3時間44分。副業の時給換算で1案件分の利益が消えていた。

## 丸投げが成立した理由：Claude Sonnet へ渡す「制約語」の有無

失敗したプロンプトと成功したプロンプトの差は1行ではない。**制約語の密度**だ。

```
# NG：制約なし（ESLint 8が返ってくる）
ESLintとPrettierの設定ファイルを作って

# OK：黄金フレーム
技術スタック: React 18 + TypeScript 5 + Vite
制約: ESLint 9 flat config、Prettier 3との競合なし、airbnb-base不使用
出力: package.json（devDepsのみ）, eslint.config.js, .prettierrc の3ファイルのみ
バージョン: npm最新stable（2024-12時点）
```

「出力ファイルを3つに限定」「バージョンをstableに固定」の2行が28分の鍵だった。制約を省くとモデルは自由に解釈し、ESLint 8のlegacy config・airbnb全部入り・不要な依存を返す。

## 無料配布：package.json / eslint.config.js / .prettierrc 一括生成スクリプト（Node.js 22行）

`setup.mjs` として保存して `node setup.mjs` を実行すると、Claude APIが3ファイルを生成して書き出す。1実行のAPIコストは**$0.003〜0.006**（2024-12月実測、claude-sonnet-4-6）。

```js
// setup.mjs
import Anthropic from "@anthropic-ai/sdk";
import { writeFileSync } from "fs";

const client = new Anthropic(); // ANTHROPIC_API_KEY を env から読み込み

const PROMPT = `技術スタック: React 18 + TypeScript 5 + Vite
制約: ESLint 9 flat config、Prettier 3との競合なし、airbnb-base不使用
出力形式: 各ファイルを「=== ファイル名 ===」で区切り本文のみ出力
対象: package.json（devDepsのみ）, eslint.config.js, .prettierrc`;

const res = await client.messages.create({
  model: "claude-sonnet-4-6",
  max_tokens: 1024,
  messages: [{ role: "user", content: PROMPT }],
});

const blocks = res.content[0].text.split(/^=== (.+) ===$/m).slice(1);
for (let i = 0; i < blocks.length; i += 2) {
  writeFileSync(blocks[i].trim(), blocks[i + 1].trim());
  console.log(`✓ ${blocks[i].trim()}`);
}
```

実行後30秒以内に3ファイルが生成される。`npm install && npm run dev` が通れば初動完了だ。
