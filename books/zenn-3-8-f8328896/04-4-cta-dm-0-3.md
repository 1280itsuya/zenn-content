## プロフィール文3版のプロフ遷移→問い合わせCVRを実数で比較

結論から言うと、問い合わせを0→月3件に増やした最大要因はプロフ末尾1行の「受託の空き枠」明示で、CVRが0.4%→2.1%へ跳ねた。3版のABテスト実数は以下。

| 版 | プロフ遷移 | 問い合わせ | CVR |
|----|----------|----------|------|
| v1 経歴羅列 | 412 | 2 | 0.49% |
| v2 実績数値 | 388 | 5 | 1.29% |
| v3 空き枠+単価明示 | 357 | 7 | **1.96%** |

検索キーワード「Claude 受託 問い合わせ」で流入した記事のCTAクリック率は、H2直後埋め込みが3.8%、記事末尾が1.2%。スクロール到達率42%が末尾の上限だった。

## Claudeでプロフ文を量産しUTMで流入計測する自動化フロー

Claude APIでプロフ案を10本生成し、UTM付きリンクで版別の到達を分離した。

```python
import anthropic, urllib.parse
client = anthropic.Anthropic()
def gen_profiles(variant: str) -> str:
    msg = client.messages.create(
        model="claude-opus-4-8", max_tokens=300,
        messages=[{"role": "user",
            "content": f"受託エンジニアのZennプロフ文。空き枠と単価帯を末尾1行で明示。版={variant}"}])
    return msg.content[0].text

base = "https://forms.example.com/contact"
link = f"{base}?{urllib.parse.urlencode({'utm_source':'zenn','utm_content':'v3'})}"
```

## 24時間以内テンプレ返信が成約率を0%→43%に上げた検証

返信が24時間超だと成約0/3、24時間以内だと成約3/7（43%）。初回返信を固定テンプレで即送る運用に切り替えた。

```python
TEMPLATE = """ご連絡ありがとうございます。
・想定要件と希望納期を3行で
・現状の月額/予算レンジ
上記を頂ければ24h以内に概算見積（月8万〜）をお送りします。"""

def auto_reply(dm) -> dict:
    return {"to": dm["sender"], "body": TEMPLATE, "sla_hours": 24}
```

プロフv3＋H2直後CTA＋X DM導線＋24h返信の4点を同時に回した月、受託単価は月3万→8万に乗り、問い合わせは月3件で安定した。
