## ban_words 辞書 420 語の構造と読み込み（YAML 分類）

商標・作家名・スタジオ名を 8 カテゴリに分け、`data/ban_words.yaml` に 420 語を保持する。半年で通報を受けた語を起点に、類義・英語表記・かな表記を手で追加した実データだ。

```yaml
studio:   ["ジブリ", "ghibli", "ディズニー", "disney", "ピクサー", "pixar"]
artist:   ["新海誠", "村上隆", "greg rutkowski", "artgerm"]
character:["ピカチュウ", "pikachu", "マリオ", "mario", "キティ"]
style_kw: ["in the style of", "風", "inspired by", "homage to"]
```

```python
import yaml, re
BAN = yaml.safe_load(open("data/ban_words.yaml", encoding="utf-8"))
FLAT = {w.lower() for cat in BAN.values() for w in cat}
```

`FLAT` は 420 要素の集合になり、後段の検出はすべて O(1) で引ける。

## sanitize_prompt() による語単位の検出と置換

完全一致だけでは「ジブリ風の」のような連結を取りこぼす。正規表現で語境界を作り、ヒット語を中立表現へ置換する。検出時は `flagged` に残してログ化する。

```python
REPLACE = {"ジブリ": "やわらかい水彩", "disney": "rounded 3D cartoon",
           "新海誠": "高彩度の空と光", "in the style of": "inspired tone"}

def sanitize_prompt(prompt: str):
    flagged, out = [], prompt
    for w in sorted(FLAT, key=len, reverse=True):
        pat = re.compile(re.escape(w), re.IGNORECASE)
        if pat.search(out):
            flagged.append(w)
            out = pat.sub(REPLACE.get(w, "generic illustration"), out)
    return out, flagged
```

長い語から処理することで「disney princess」を「disney」断片より先に潰す。日 500 枚で平均 1.8% がここで書き換わった。

## LoRA と `<lora:...>` 構文の特殊検出

SDXL の `<lora:ghibli_style:0.8>` はプロンプト本文の辞書照合をすり抜ける。専用パターンで重み付きトリガーを抜き出し、権利系 LoRA 名を遮断する。

```python
LORA_DENY = {"ghibli", "makoto", "disney", "studio"}

def strip_risky_lora(p: str):
    def repl(m):
        name = m.group(1).lower()
        return "" if any(k in name for k in LORA_DENY) else m.group(0)
    return re.sub(r"<lora:([^:>]+):[\d.]+>", repl, p)
```

420 語辞書を通過した後でも、LoRA 名由来の権利通報が月 2 件残っていた。この遮断を入れた翌月から該当 0 件になった。

## Claude Haiku による固有名詞の二重チェック

辞書は既知語しか潰せない。新作アニメ名など未収録の固有名詞は Claude Haiku（claude-haiku-4-5）に「商標・作品名を一般的な画風語へ言い換える」よう指示し、JSON で受け取る。1 リクエスト約 ¥0.4、日 500 枚で月 ¥6,000 弱に収まる。

```python
import anthropic, json
cli = anthropic.Anthropic()

def llm_guard(prompt: str) -> str:
    r = cli.messages.create(
        model="claude-haiku-4-5",
        max_tokens=300,
        messages=[{"role": "user", "content":
            f"次の画像生成プロンプトから商標・作品名・作家名を一般的な"
            f"画風表現へ置換し、JSON {{\"clean\": str}} のみ返せ:\n{prompt}"}])
    return json.loads(r.content[0].text)["clean"]
```

辞書 → LoRA 除去 → Haiku の 3 段で、未収録語の漏れを実測 0.3% まで下げた。
