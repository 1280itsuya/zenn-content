## $0.59/日の内訳をトークンで分解する

コストの正体は Haiku の入力トークンだ。1,284件×平均60トークンを4回巡回した実数で計算する。

```python
TWEETS, TOK_IN, RUNS = 1284, 60, 4
PRICE_IN = 0.80 / 1_000_000   # Haiku 3.5 入力 $0.80/1M tokens
cost = TWEETS * TOK_IN * RUNS * PRICE_IN
print(f"${cost:.2f}/day")     # -> $0.25/day (分類のみ)
# 要約配信のClaude出力分を足して実測 $0.59/day
```

## 評価スクリプトでミュート精度93%を再現する

「だいたい合ってる」では運用に乗らない。手ラベル200件と突き合わせ、precision を数値で固定する。

```python
import json
from sklearn.metrics import precision_score

pred = [r["label"] for r in json.load(open("out.json"))]
true = [r["gold"]  for r in json.load(open("labeled_200.json"))]
mute_p = precision_score(true, pred, labels=["mute"], average="micro")
print(f"mute precision: {mute_p:.2f}")   # -> 0.93
```
