## POST /2/users/:id/muting——月500回クォータの管理コード

無料枠の生命線はこのエンドポイント1本に集約される。月500回を超えた瞬間に429が返り、雑に作ると月初までBotが沈黙する。筆者はこれで3回停止させた(3月14日・3月22日・4月2日、停止ログは第5章に掲載)。対策はローカルでのクォータ事前カウントだ。

```python
def remaining_quota(db: sqlite3.Connection) -> int:
    month = datetime.now(timezone.utc).strftime("%Y-%m")
    used = db.execute(
        "SELECT COUNT(*) FROM muted WHERE at LIKE ?", (f"{month}%",)
    ).fetchone()[0]
    return MUTE_QUOTA_PER_MONTH - used   # 残量が0ならAPIを叩く前に止まる
```

429を受けてから待つのではなく、叩く前に残量を見る。この4行を入れてから90日間、レート起因の停止はゼロになった。
