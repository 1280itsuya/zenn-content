## リコール7件の発生源は3つに分解できる

Pinterestから届いた著作権リコール7件をログで突き合わせると、発生源はプロンプト直書きの固有名詞4件、merge LoRAの学習痕跡2件、生成物の構図一致1件に分かれた。danbooruタグ由来のキャラ名がプロンプトに混入し、SDXL Turboがそのキャラを再現してしまうケースが過半数を占める。

```python
recall_log = {"prompt_ngword": 4, "lora_metadata": 2, "clip_similarity": 1}
total = sum(recall_log.values())  # 7
print({k: round(v/total, 2) for k, v in recall_log.items()})
# {'prompt_ngword': 0.57, 'lora_metadata': 0.29, 'clip_similarity': 0.14}
```

## プロンプトNGワード辞書430語で投入前に弾く

実在キャラ・商標・既存作家名を430語の辞書にまとめ、生成リクエスト前に正規化マッチで遮断する。表記揺れ（全角/半角、`_`区切りのdanbooru形式）を吸収するため小文字化とアンダースコア展開を噛ませる。これで発生源の57%を投入前に潰せる。

```python
import json, re

NG = set(json.load(open("ngwords_430.json")))  # 実在キャラ/商標/作家名 430語

def is_clean(prompt: str) -> tuple[bool, list[str]]:
    tokens = re.split(r"[,\s_]+", prompt.lower())
    hits = [t for t in tokens if t in NG]
    return (len(hits) == 0, hits)

ok, hits = is_clean("1girl, hatsune_miku, blue twintails")
print(ok, hits)  # False ['hatsune_miku']
```

## merge LoRAのss_tag_frequencyで学習痕跡を炙り出す

配布merge LoRAは学習元を隠していても、メタデータ`ss_tag_frequency`に元タグの出現頻度が残る。safetensorsのヘッダだけ読めばGPUに載せずに走査でき、NG辞書とヒットしたLoRAは採用前に除外する。2件の汚染LoRAはこの走査で事前に検出できた。

```python
import json, struct

def scan_lora(path: str) -> list[str]:
    with open(path, "rb") as f:
        n = struct.unpack("<Q", f.read(8))[0]
        meta = json.loads(f.read(n)).get("__metadata__", {})
    freq = json.loads(meta.get("ss_tag_frequency", "{}"))
    tags = {t.lower() for d in freq.values() for t in d}
    return sorted(tags & NG)

print(scan_lora("anime_merge_v3.safetensors"))  # ['genshin_impact', 'nahida']
```

## CLIP類似度0.9超を事後フィルタで除去する

プロンプトとLoRAが綺麗でも、構図が既存作品と一致すれば残り14%のリコールになる。生成画像と参照作品800枚をCLIP（ViT-B/32）で埋め込み、コサイン類似度0.9超を破棄する。閾値0.9は手動レビュー120枚でFP（誤除去）を6%に抑えた値。

```python
import torch, clip
from PIL import Image

model, prep = clip.load("ViT-B/32")
refs = torch.load("ref_embeds_800.pt")  # [800, 512] 既存作品

def reject(img_path: str, thr: float = 0.9) -> bool:
    img = prep(Image.open(img_path)).unsqueeze(0)
    with torch.no_grad():
        e = torch.nn.functional.normalize(model.encode_image(img).float())
    return (e @ refs.T).max().item() > thr

print(reject("out/0421.png"))  # True なら投稿しない
```

## 月7件→0件にした三段ゲート運用フロー

辞書遮断→LoRA走査→CLIP事後フィルタを直列に並べ、1つでも引っかかった生成物は`quarantine/`へ隔離する。導入後3か月でPinterestリコールは月7件から0件、隔離率は全生成2,400枚中の3.1%（74枚）に収まった。隔離ログは辞書の追補にそのまま回す。

```python
def gate(prompt, lora, img):
    if not is_clean(prompt)[0]:      return "ng:prompt"
    if scan_lora(lora):              return "ng:lora"
    if reject(img):                  return "ng:clip"
    return "pass"

stats = {"pass": 2326, "ng:prompt": 41, "ng:lora": 9, "ng:clip": 24}
quarantined = sum(v for k, v in stats.items() if k != "pass")
print(quarantined, round(quarantined / sum(stats.values()) * 100, 1))  # 74 3.1
```
