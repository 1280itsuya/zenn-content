## AIが「勝手に稼ぐ」とはどういう意味か

「AIが稼ぐ」というのは、あなたが寝ている間にAIが文章を書き、それがWeb上に公開され、アクセスを集め、収益につながる状態を指す。魔法ではなく、**入力→生成→投稿→収益化**の4ステップを自動でつなぐだけだ。

核心はシンプルだ。AIに「何を書くか」を決めさせ、「書かせ」、「どこに出すか」まで実行させる。あなたの仕事は、最初に仕組みを組み立てることだけになる。

---

## 最速で動かす3ステップ

### ステップ1：Claude CLIをインストールする

```bash
npm install -g @anthropic-ai/claude-code
claude --version
```

Anthropic公式のCLIを使う理由は、`claude -p "プロンプト"`の一行でAI生成をスクリプトから呼び出せるからだ。APIキーなしで使えるMaxプランが特にコスパが高い。

### ステップ2：生成→投稿スクリプトを書く

以下は「技術記事を1本生成してZennに投稿する」最小構成のPythonスクリプトだ。

```python
import subprocess, pathlib, datetime

topic = "Claude APIでブログを自動生成する方法"
prompt = f"{topic}について、初心者向けに1000字の技術記事を書いてください。"

result = subprocess.run(
    ["claude", "-p", prompt],
    capture_output=True, text=True
)
article = result.stdout

slug = datetime.date.today().strftime("%Y%m%d") + "-auto"
path = pathlib.Path(f"articles/{slug}.md")
path.write_text(f"---\ntitle: {topic}\nemoji: 🤖\ntype: tech\ntopics: [claude, 自動化]\npublished: true\n---\n\n{article}")

subprocess.run(["git", "add", ".", "--all"])
subprocess.run(["git", "commit", "-m", f"auto: {slug}"])
subprocess.run(["git", "push"])
```

Zennはリポジトリと連携すれば`git push`だけで公開できる。このスクリプトを保存したら、あとはスケジュール実行に登録するだけだ。

### ステップ3：毎朝7時に自動実行する

Windowsならタスクスケジューラ、Macならcronに登録する。

```
# crontab -e に追加（Mac/Linux）
0 7 * * * python3 /home/user/auto-blog/run.py >> /tmp/auto-blog.log 2>&1
```

これで毎朝7時に記事が1本自動生成・公開される。月に30本の記事が、あなたの手を一切借りずに積み上がる。

## 収益までの最短経路

記事が公開されただけでは収益にならない。**流入→クリック→報酬**の導線が必要だ。最速の構成はこうだ。

1. **Zenn/Qiita**に技術記事を投稿（プラットフォームの既存読者を借りる）
2. 記事末尾に**Gumroad商品やアフィリリンク**を埋め込む
3. 記事が検索・おすすめに露出するとクリックが発生し収益になる

自前サイトでSEOを育てるより、読者がすでにいるプラットフォームに乗るほうが圧倒的に早い。最初の収益は「自分のサイト」ではなく「他人の流入」から生まれる。

---

## 本章のまとめ

仕組みは3日で動く。CLIのインストール、スクリプト1本、スケジューラへの登録だけだ。大事なのは「完璧な記事」を狙わないことで、**毎日1本・自動・公開済み**の状態を最速で作ることに集中する。動き始めた仕組みは、あとから改善できる。止まったままの計画は、何も生まない。
