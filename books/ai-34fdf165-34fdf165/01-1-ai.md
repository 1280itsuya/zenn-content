## AIを「道具」で止めるな

ChatGPTで文章を書いてもらった。画像生成AIで素材を作った。それで満足していないか。

「使う」と「働かせる」は根本的に違う。使うのは人間が毎回操作するモデル。働かせるのはAIが**自律的に動き続ける**モデルだ。前者は自分の時間を消費する。後者は自分が寝ている間にアウトプットが積み上がる。

月収を変えるのは後者しかない。

---

## 「働かせる」とは何か

具体的に見せよう。以下は筆者が実際に動かしているPythonスクリプトの骨格だ。

```python
# orchestrator.py（毎朝7時にタスクスケジューラから起動）
from agents import blog_writer, zenn_poster, affiliate_injector

topics = generate_topics(niche="節約術", count=3)

for topic in topics:
    article = blog_writer.run(topic)
    article = affiliate_injector.run(article)
    zenn_poster.publish(article)
```

このスクリプトは毎朝7時に自動起動し、ニッチを決めてテーマを立案し、記事を生成し、アフィリエイトリンクを挿入し、Zennに投稿する。人間がやることはゼロだ。

タスクスケジューラへの登録はWindows PowerShellで一発だ。

```powershell
$action = New-ScheduledTaskAction -Execute "python" `
    -Argument "C:\auto-money\orchestrator.py"
$trigger = New-ScheduledTaskTrigger -Daily -At 7am
Register-ScheduledTask -Action $action -Trigger $trigger -TaskName "AutoMoney"
```

これで「毎朝7時に自動実行」が完成する。

## 「使う側」の天井

「使う側」には構造的な天井がある。1時間で処理できる量は人間の集中力に縛られる。体調が悪ければアウトプットはゼロだ。

対して「働かせる側」は違う。

- 記事生成：1日3本 × 30日 ＝ 90本（自動）
- Zenn投稿：push一発で公開（自動）
- アフィリリンク挿入：テンプレートで即注入（自動）

この差が6ヶ月後に数十万円の差になる。AIは疲れない。休まない。コストは1記事あたりAPI費用で数円だ。

---

## 最初の一歩：単一タスクを自動化する

いきなり全部自動化しようとすると詰まる。まず**1つのタスクだけ**をスクリプト化せよ。

```bash
# ステップ1：OpenAI APIでタイトル候補を生成する
python scripts/generate_titles.py --niche "副業" --count 5
```

動いたら次のステップを繋ぐ。記事本文生成 → 投稿 → レポート通知、と1ステップずつパイプラインを伸ばす。

本書はこの「パイプラインを育てる」思考をすべての章で貫く。AIを「使う」習慣を捨て、AIを「組む」習慣に切り替える。それがこの本の出発点だ。
