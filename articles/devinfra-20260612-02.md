# 自動化botを動かすレンタルサーバー選びの結論と実務手順

## 結論：[個人開発](https://www.amazon.co.jp/s?k=%E5%80%8B%E4%BA%BA%E9%96%8B%E7%99%BA%20%E6%9C%AC&tag=1280itsuya22-22)の常駐botなら「格安VPS 1〜2GBプラン」一択

毎朝のスクレイピング、定期投稿、cron駆動の自動化スクリプトを24時間動かすなら、共有レンタルサーバーやサーバーレスではなく **root権限のあるVPS** を選ぶべきです。理由はこの3点に尽きます。

- **常駐プロセスとcronが自由に組める** … 共有サーバーは長時間プロセス・SSH・任意パッケージ導入に制限があり、botとは相性が悪い
- **環境を汚さず使い捨てできる** … `apt`やDocker、[Python](https://www.amazon.co.jp/s?k=Python%20%E5%85%A5%E9%96%80%20%E6%9C%AC&tag=1280itsuya22-22) venv、Node、ヘッドレスChromeまで自前で入れられる
- **固定IPで外部APIのIP制限・SSH運用が安定する**

候補としては **ConoHa VPS / Xserver VPS / さくらのVPS** あたりの国内VPS、[ドメイン](https://px.a8.net/svt/ejp?a8mat=4B3XB4+4VMW8I+50+2HU3GX)を併せて取るなら **お名前.com** が現実的です。料金は各社とも頻繁に改定されるため、**必ず公式の最新価格を確認してください**（以下は参考レンジで、確定値ではありません）。

## 根拠：botに必要なスペックは意外と小さい

定期実行のbotはCPUを常時食うわけではなく、**メモリが律速**になりがちです。Pythonの`requests`中心なら512MB〜1GBでも動きますが、**Selenium/Playwrightでブラウザを立てる用途は最低2GB** を見ておくと安定します。

- メモリ1GBプラン … 軽量なAPI連携・cron投稿bot向け（参考: 月額1,000円前後のレンジ。要公式確認）
- メモリ2GBプラン … ヘッドレスブラウザ・複数bot同居向け
- ストレージはSSD 30GB前後あればログ込みで十分

長期割引（12ヶ月一括など）で単価が下がる一方、解約時の扱いは各社で違うので、**最初は最短契約で試す**のが無難です。

## 具体手順：契約からbot常駐まで

### 1. VPS契約とOS選択
OSは扱い慣れた **Ubuntu LTS** を推奨。契約直後にrootパスワード or SSH鍵が発行されます。

### 2. SSH鍵でログイン（パスワード認証は早めに無効化）

```bash
ssh-keygen -t ed25519 -C "bot-server"
ssh-copy-id -i ~/.ssh/id_ed25519.pub user@<VPSのIP>
```

`/etc/ssh/sshd_config` で `PasswordAuthentication no` にしておくと総当たり攻撃をほぼ無効化できます。

### 3. 実行環境の構築

```bash
sudo apt update && sudo apt install -y python3-venv git
git clone <your-repo> && cd <your-repo>
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
```

### 4. cronで定期実行

```bash
crontab -e
# 毎朝7時に実行（パスは絶対指定が安全）
0 7 * * * cd /home/user/bot && /home/user/bot/.venv/bin/python main.py >> log.txt 2>&1
```

常駐型なら `systemd` のサービス化で自動再起動を付けると堅牢です。

### 5. DNS（独自ドメインを使う場合）
お名前.com等で取得したドメインを使うなら、VPSのIPを **Aレコード** に登録します。

```
A  bot.example.com  <VPSのIP>  TTL 3600
```

反映には数分〜最大48時間かかることがあります。webhook受け口やステータスページを公開する時に必要になりますが、**送信専用botならDNSは不要**です。

## まとめ

botサーバー選びは「**root権限・SSH・cronが自由に使えるVPSを、必要メモリ（軽量1GB／ブラウザ2GB）で選ぶ**」がほぼ唯一の正解です。料金は変動するため公式で最新を確認し、まずは最短契約・SSH鍵運用から始めてください。
