# AI副業の月次インフラ原価を晒す：個人開発者が実際に払っているもの

## 結論から言う

AI系の副業自動化パイプラインを回すのに、毎月かかるインフラ費用の実態を公開する。「思ったより安い」と感じる人もいれば「こんなにかかるのか」と驚く人もいるだろう。いずれにせよ、数字を見ないまま「なんとなく運用している」状態は危険だ。コスト管理は収益化の前提条件である。

## 月次コストの内訳

### 1. VPS：自動化の土台

自宅PCだけで完結させようとすると、停電・再起動・ISP切断でジョブが飛ぶ。最終的にVPSを1台立てて「常駐ジョブの番人」にした。

現時点で私が使っている選択肢の比較：

- **ConoHa VPS**（1GBプラン）：公式サイト掲載の定価ベースで例: 月額880円前後。時間課金もあり、検証用途で一時的に立てやすい。コントロールパネルが日本語で整っていてSSH鍵のGUI登録がスムーズ。
- **Xserver VPS**（スタンダードプラン）：例: 月額830円前後（キャンペーン時はさらに安い場合あり）。Xserverのアカウントを既に持っている場合、請求が一本化できて管理が楽。
- **さくらのVPS**（512MBプラン）：老舗で安定性の評判が高い。石橋を叩きたい人向け。

**重要：上記料金はすべて変動します。申し込み時に公式サイトで必ず確認してください。**

### 2. ドメイン：SEOと信頼性の投資

独自ドメインは `.com` か `.dev` が無難。お名前.com で取得しているが、初年度格安→更新で跳ね上がるパターンに注意。例: 年額1,500円〜2,000円前後で安定しているケースが多いが、TLDや取得サービスによって大きく異なる。月割りにすると例: 月額150円前後のイメージ。

### 3. LLM API：最大の変動費

AI副業の文脈では、ここが一番のコスト変動要因になる。私は[Claude](https://www.anthropic.com/) CLIのMax定額プランに切り替えることで予測可能なコストに固定した。従量課金のままだと、自動化ループがバグった瞬間に予想外の請求が来る。**自動実行ジョブにはかならず月間上限・アラートを設定すること。**

## SSH・DNS設定の実務ポイント

### SSH鍵認証は絶対

パスワード認証は即座に無効化する。以下の手順：

```bash
# 公開鍵をサーバーに転送
ssh-copy-id -i ~/.ssh/id_ed25519.pub user@your-vps-ip

# sshd_config を編集
sudo nano /etc/ssh/sshd_config
# PasswordAuthentication no
# PermitRootLogin no

sudo systemctl restart sshd
```

ポートも22から変える派と変えない派がいるが、fail2banを入れるほうが本質的な対策になる。

### DNS：お名前.comのAレコード設定

VPSのIPが変わった際にDNS更新を忘れると全サービスが死ぬ。作業手順：

1. VPSのダッシュボードでIPアドレスを確認
2. お名前.com管理画面 → DNS設定 → Aレコードに登録
3. TTLは3600秒（1時間）を基本に。切り替え作業前は300秒に下げると反映が速い
4. `dig yourdomain.com` で伝播を確認してから本番向け

## コスト削減で実際に効いた施策

**1. 自宅PCをメイン実行機にする**
VPSはWebhook受信とモニタリング専用に絞り、重い処理（LLM呼び出し）は自宅PCで動かすことで、VPSのプランを最小構成に落とせる。

**2. 無料枠の活用**
GitHub Actions（月2,000分無料）、Cloudflare（DNS・CDN無料プラン）を使えばVPSへの依存度を下げられる。ただし無料枠は突然変更される可能性がある点は覚悟しておく。

**3. 定期的なコスト棚卸し**
毎月末に請求明細を確認して「使っていないリソース」を即削除する習慣をつける。スナップショット・固定IPの放置は地味にコストを積み上げる。

---

## まとめ：月次インフラ原価の現実

私のケースでは、VPS＋ドメイン＋その他ツールを合算すると例: 月額1,500〜2,500円前後に収まっている。LLM APIをMax定額プランで固定したことで予算が読みやすくなった。収益がゼロのうちはこのコストが純損失になるため、**「動いているか」より「集客できているか」を先に検証する**のが正しい順番だと実感している。

インフラを整えることは必要条件だが十分条件ではない。コストを可視化した上で、投資対効果を冷静に判断し続けることが個人開発の継続性につながる。
