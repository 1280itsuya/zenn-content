## 結論から先に

個人開発でVPSを選ぶとき、カタログスペックより「毎月の実費」「SSHの体感速度」「帯域制限でハマらないか」の3点が実務に直結する。筆者はConoHa VPS・Xserver VPS・さくらのVPSを同一用途（Node.js APIサーバ＋cronジョブ）で使い比べた経験から整理する。

**先に結論：**
- **コスパ＆短期検証ならConoHa VPS**（時間課金＋月額上限）
- **安定運用＋サポート重視ならXserver VPS**（長期割が強い）
- **長期・固定費最小化ならさくらのVPS**（老舗の安定感）

---

## 1. 料金の実態

VPS料金は各社が頻繁に改定するため、**以下はすべて参考値であり、契約前に公式サイトで必ず確認すること。**

### ConoHa VPS

時間課金（例: 1GBプランで1時間数円台）と月額上限制の組み合わせが特徴。試作環境を「使うときだけ立ち上げる」用途では、月額固定契約より割安になるケースがある。コントロールパネルからサーバ追加・削除が即時できるため、スケールアップ検証にも向く。

### Xserver VPS

月額固定で例: 1GBプランが数百円台から。12ヶ月・36ヶ月の長期契約で割引率が大きくなる構造。「このサーバを数年使い続ける」と決めているなら、長期契約の総額はConoHaの月次より安くなることが多い。

### さくらのVPS

料金体系がシンプルで老舗ゆえの安定感がある。国内複数リージョン（東京・石狩など）を選べる。お名前.com VPSも競合として名前が挙がるが、料金改定が頻繁なため今回は参考にとどめる。

---

## 2. SSH応答の体感

計測は国内リージョンのみ（東京・大阪）を対象に、自宅PCから公開鍵認証でSSHログインした際の主観評価。絶対値ベンチマークは計測環境に依存しすぎるため数値提示はしない。

- **ConoHa（東京）**: 接続確立が体感的に速く、vim・tmuxの操作でストレスを感じる場面はほぼなかった。
- **Xserver VPS**: 業務用途では問題ないが、大量ファイルをscpする際に若干待ち時間が気になる場面があった。
- **さくらのVPS（石狩）**: 東京リージョン比で遅延が体感できる。東京リージョンを選べば実用上の問題はない。

**重要な教訓：SSHの速度感は接続先リージョンが支配的。** ベンチマーク数値より「自分の作業拠点に近いリージョンがあるか」を先に確認するほうが実態に近い。

### 公開鍵認証の最短セットアップ

```bash
# ローカルで鍵生成（Ed25519推奨）
ssh-keygen -t ed25519 -C "myserver"

# サーバ側に配置
ssh-copy-id -i ~/.ssh/id_ed25519.pub user@<VPS_IP>

# パスワード認証を無効化（/etc/ssh/sshd_config）
PasswordAuthentication no
PubkeyAuthentication yes

sudo systemctl restart sshd
```

## 3. 転送量の扱い

3社とも個人用途では「帯域超過で課金爆発する」モデルではなく、月額固定内で収まる設計になっている。動画配信や大容量ファイル配布を想定するなら、各社の帯域制限ポリシー（利用規約に記載）を契約前に読んでおくこと。

---

## 4. 選び方フローとDNS設定

```
用途を確認
├─ 短期・検証・PoC         → ConoHa VPS（時間課金で損失最小）
├─ 1〜3年の安定運用        → Xserver VPS（長期割＋サポート）
└─ 長期 or コスト最小化    → さくらのVPS（シンプルな料金体系）
```

[ドメイン](https://px.a8.net/svt/ejp?a8mat=4B3XB4+4VMW8I+50+2HU3GX)をお名前.comで管理している場合、VPSのIPへのAレコード設定はコントロールパネルから5分以内に完了する。ネームサーバをお名前.comのまま使うか、VPS側のDNSに移すかは、DNS更新頻度と管理の一元化で判断する。

| 観点 | ConoHa | Xserver VPS | さくらのVPS |
|---|---|---|---|
| 料金柔軟性 | ◎（時間課金） | △（長期割必要） | ○（長期割あり） |

**料金は常に変動する。** 最終判断は各社の公式料金ページを確認した上で行ってほしい。「まず1ヶ月試す」ならConoHaの時間課金が最も損失が少なく、長期運用が決まっているならXserver VPSかさくらのVPSの長期契約を比較するのが合理的な判断順序だ。
